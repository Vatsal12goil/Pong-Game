# Pong Game

## Description
Pong Game is a classic arcade-style game that simulates table tennis where two players compete against each other. The objective of the game is to score points by hitting a ball back and forth across a screen. Players control paddles to bounce the ball back to their opponent, aiming to make the ball pass their opponent's paddle to score a point.

## Features
- **Multiplayer**: Play against a friend on the same device.
- **Simple Controls**: Intuitive controls for paddle movement.
- **Score Tracking**: Keep track of the score for each player.
- **AI Opponent**: Play against a computer-controlled opponent.
- **Customizable Settings**: Adjust the game speed and difficulty.

## How to Play
1. Launch the game.
2. Use the controls to move your paddle up and down.
3. Try to hit the ball back to your opponent's side of the screen.
4. If the ball passes your paddle, the opposing player scores a point.
5. Win by reaching the designated number of points.

## Technical Details
- **Language**: JavaScript
- **Libraries**: This game utilizes HTML5 Canvas for rendering graphics.
- **Setup**: Clone the repository and open `index.html` in your web browser to start playing.
